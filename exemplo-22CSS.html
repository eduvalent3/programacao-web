O que é o Grid Latoult

Flexgrid ou Grid Layout

O flexbox e Grid Layout podem ser utilizados em conjuntos. Você pode aplicar o Flexbox dentro de um layout que esteja usando o Grid Layout, aproveitando o melhor das duas técnicas.

A escolha entre Flexbox e Grid Layout não se trata de um substituir o outro, mas sim de entender as necessidades específicas de cada layout aplicando as suas necessidades.
