Listas Ordenadas(ol) e Listas não Ordenadas(ul)

Em html, os elementos de mlista são usados para criar listas de intes organizados em uma página web

Lista ordenada -ol

para criar uma lista ordenada no html eu devo utilizar a tag ol, juntamente com li, que representa o elementos da lista. exemplos:

  1. primerio item
    1. Item 1.1
    2. Item 1.2
      1. Item 1.2.1
      2. Item 1.2.2
    3. Item 1.3
  2. segundo item
  3. terceiro item
  4. quarto item
  5. quinto item
  6. sexto item

Lista não ordenada -ul

para criar uma lista não rdenada no html eu devo utilizar a tag ol, juntamente com li, que representa o elementos da lista. exemplos:

  • primerio item
    • Item 1.1
    • Item 1.2
      • Item 1.2.1
      • Item 1.2.2
    • Item 1.3
  • segundo item
  • terceiro item
  • quarto item
  • quinto item
  • sexto item
